001 : PDF Merger

The PDF Merger is a straightforward yet powerful tool designed to combine multiple PDF documents into a single, cohesive file. This project showcases my proficiency in handling file operations and streamlining document management processes. It's particularly useful for anyone who needs to merge reports, forms, or any other PDF files efficiently, making it an essential utility for both personal and professional use.


      import os
      from PyPDF2 import PdfWriter, PdfMerger

      merger = PdfMerger()
      inp = list()

      pdfFiles = []  # variable

      directory = './pdf_files'
      for filename in os.listdir(directory):
          pdfFiles.append(f'{directory}/{filename}')

      pdfFiles.sort()
      pdfFiles.pop(0)
      print(pdfFiles)
      for pdf in pdfFiles:
          merger.append(pdf)

      with open("merged-pdf.pdf", "wb") as new_file :
          merger.write(new_file)
